## 📌 4013 특이한 자석

### 💡 핵심 아이디어
- 회전 전파 방향 먼저 기록(`rot[]`), 그 다음 **한 번에** 전부 회전
- 왼쪽 전파: `arr[i][6]` (오른쪽 날) vs `arr[i-1][2]` (왼쪽 이웃 왼쪽 날)
- 오른쪽 전파: `arr[i][2]` vs `arr[i+1][6]`
- 다를 때만 반대 방향으로 연쇄 전파, 같으면 전파 중단

```python
for _ in range(K):
    num,d = map(int, input().split())
    num -= 1
    rot = [0]*4
    rot[num] = d

    # 왼쪽 전파
    for i in range(num, 0, -1):
        if arr[i][6] != arr[i-1][2]: rot[i-1] = -rot[i]
        else: break

    # 오른쪽 전파
    for i in range(num, 3):
        if arr[i][2] != arr[i+1][6]: rot[i+1] = -rot[i]
        else: break

    # 한 번에 회전
    for i in range(4):
        if   rot[i]==1:  arr[i]=[arr[i][-1]]+arr[i][:-1]   # 시계
        elif rot[i]==-1: arr[i]=arr[i][1:]+[arr[i][0]]     # 반시계

# 점수: 12시(인덱스 0)가 S극(1)이면 득점
ans = sum((arr[i][0]==1)*(2**i) for i in range(4))
```

## 📌 4014 활주로 건설

### 💡 핵심 아이디어
- 각 행/열을 독립적으로 `check()` 함수로 검사
- 오르막: **뒤쪽** X칸이 같은 높이인지 확인 후 경사로 표시
- 내리막: **앞쪽** X칸이 같은 높이인지 확인 후 경사로 표시 + i를 X만큼 점프
- `used[]` 배열로 경사로 중복 설치 방지

```python
def check(arr, X):
    N = len(arr)
    used = [0]*N
    i = 0
    while i < N-1:
        if arr[i]==arr[i+1]:
            i+=1
        elif arr[i]+1==arr[i+1]:       # 오르막: 뒤쪽 X칸 확인
            for j in range(i, i-X, -1):
                if j<0 or arr[j]!=arr[i] or used[j]: return False
                used[j]=1
            i+=1
        elif arr[i]-1==arr[i+1]:       # 내리막: 앞쪽 X칸 확인
            for j in range(i+1, i+X+1):
                if j>=N or arr[j]!=arr[i+1] or used[j]: return False
                used[j]=1
            i+=X
        else: return False
    return True

for i in range(N):
    if check(board[i], X): ans+=1
    col = [board[j][i] for j in range(N)]
    if check(col, X): ans+=1
```

## 📌 1949 등산로 조성

### 💡 핵심 아이디어
- 최고봉에서 출발, **높은 곳 → 낮은 곳**으로만 이동 (DFS)
- 딱 **한 번** 깎기 가능: `cut` 플래그로 사용 여부 추적
- 깎을 때: `arr[ni][nj] = arr[i][j] - 1` 로 임시 변경 후 탐색 → 복원
- 깎기 조건: `arr[ni][nj] - k < arr[i][j]` (최대 k만큼 깎으면 낮아지는지)

```python
def dfs(i,j,length,cut):
    global ans
    ans = max(ans, length)
    for dir in dirs:
        ni,nj = i+dir[0], j+dir[1]
        if 0<=ni<n and 0<=nj<n:
            # 안 깎고 진행
            if arr[ni][nj]<arr[i][j] and not visited[ni][nj]:
                visited[ni][nj]=True
                dfs(ni,nj,length+1,cut)
                visited[ni][nj]=False
            # 깎고 진행 (cut 아직 안 썼을 때만)
            elif not cut and arr[ni][nj]-k<arr[i][j] and not visited[ni][nj]:
                tmp=arr[ni][nj]
                arr[ni][nj]=arr[i][j]-1   # 임시 깎기
                visited[ni][nj]=True
                dfs(ni,nj,length+1,True)
                visited[ni][nj]=False
                arr[ni][nj]=tmp           # 복원

max_h = max(max(row) for row in arr)
for i in range(n):
    for j in range(n):
        if arr[i][j]==max_h:
            visited[i][j]=True
            dfs(i,j,1,False)
            visited[i][j]=False
```

## 📌 5658 보물상자 비밀번호

### 💡 핵심 아이디어
- `deque.rotate(1)` 로 시계방향 회전 구현
- 한 변 길이 = `n//4`, 회전 횟수 = `n//4` 번 (한 변 길이만큼만 돌리면 중복)
- 각 회전마다 4변을 16진수로 변환 → `set`으로 중복 제거
- 정렬 후 K번째로 큰 수 = `sorted[-k]`

```python
from collections import deque
nums = deque(input().strip())
side = n//4
created_nums = set()

for _ in range(side):
    arr = list(nums)
    for i in range(0, n, side):
        side_num = int(''.join(arr[i:i+side]), 16)
        created_nums.add(side_num)
    nums.rotate(1)   # 시계방향 1칸 회전

created_nums = sorted(created_nums)
print(created_nums[-k])
```

## 📌 1952 수영장 (DP)

### 💡 핵심 아이디어
- `dp[i]` = i월까지 처리하는 최소 비용
- 매월 선택지: **일일권** vs **월간권** 중 싼 것 + `dp[i-1]`
- 3개월 이상이면 **3개월권** 도 비교: `dp[i-3] + cost[2]`
- 마지막에 **연간권**과 비교

```python
days_cost = [cost[0]*month[i] for i in range(12)]  # 일일권으로만 낼 때
dp = [0]*13

for i in range(1, 13):
    dp[i] = min(days_cost[i-1], cost[1]) + dp[i-1]  # 일일 or 월간
    if i>=3:
        dp[i] = min(dp[i], dp[i-3]+cost[2])          # 3개월권

ans = min(dp[12], cost[3])   # 연간권과 최종 비교
```
