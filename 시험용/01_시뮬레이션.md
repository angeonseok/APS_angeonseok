## 📌 2382 미생물 격리

### 💡 핵심 아이디어
- 매 시간: 이동 → 가장자리 체크(절반 사망 + 방향 반전) → 위치별로 묶어 합치기
- 합칠 때 **미생물 수 가장 많은 놈의 방향** 채택
- `pos` 딕셔너리로 위치별 군집 관리, max값 별도 추적

```python
dirs = {1:(-1,0), 2:(1,0), 3:(0,-1), 4:(0,1)}
r_dirs = {1:2, 2:1, 3:4, 4:3}

for _ in range(m):
    for status in micro:
        status[0] += dirs[status[3]][0]
        status[1] += dirs[status[3]][1]
        if status[0]==0 or status[0]==n-1 or status[1]==0 or status[1]==n-1:
            status[2] //= 2
            status[3] = r_dirs[status[3]]

    pos = {}
    for status in micro:
        if status[2] == 0: continue
        key = (status[0], status[1])
        if key not in pos:
            pos[key] = [status[0], status[1], status[2], status[3], status[2]]  # r,c,합산,방향,max
        else:
            if status[2] > pos[key][4]:
                pos[key][3] = status[3]
                pos[key][4] = status[2]
            pos[key][2] += status[2]

    micro = [[v[0],v[1],v[2],v[3]] for v in pos.values()]

ans = sum(s[2] for s in micro)
```

## 📌 5644 무선 충전

### 💡 핵심 아이디어
- A, B 각자 매 시간 이동 후 접속 가능한 BC 목록 뽑기
- A가 고를 BC × B가 고를 BC의 **모든 조합 브루트포스**
- 같은 BC 선택 시 P/2 + P/2 = P이므로 그냥 P 한 번만 더함

```python
dx = [0,-1,0,1,0]
dy = [0,0,1,0,-1]

for t in range(M+1):
    ax += dx[moveA[t]]; ay += dy[moveA[t]]
    bx += dx[moveB[t]]; by += dy[moveB[t]]

    a_list, b_list = [], []
    for i in range(A):
        x,y,c,p = bc[i]
        if abs(ax-x)+abs(ay-y) <= c: a_list.append(i)
        if abs(bx-x)+abs(by-y) <= c: b_list.append(i)
    a_list.append(-1); b_list.append(-1)  # 못 쓰는 경우 대비

    best = 0
    for ai in a_list:
        for bi in b_list:
            if   ai==-1 and bi==-1: total = 0
            elif ai==-1:            total = bc[bi][3]
            elif bi==-1:            total = bc[ai][3]
            elif ai==bi:            total = bc[ai][3]   # 같은 BC → P 한 번
            else:                   total = bc[ai][3] + bc[bi][3]
            best = max(best, total)
    ans += best
```

## 📌 5648 원자 소멸 시뮬레이션

### 💡 핵심 아이디어
- **0.5초 단위 충돌**을 잡으려고 좌표를 **2배 뻥튀기**해서 시작
- 같은 위치에 2개 이상 모이면 전부 에너지 방출 후 소멸
- 범위 밖(-2000~2000) 나간 원자는 버림

```python
dx = [0,0,-1,1]
dy = [1,-1,0,0]

atoms = [[x*2, y*2, d, e] for ...]  # 좌표 2배

for _ in range(4000):
    pos = {}
    moved = []
    for x,y,d,e in atoms:
        nx,ny = x+dx[d], y+dy[d]
        if -2000<=nx<=2000 and -2000<=ny<=2000:
            moved.append([nx,ny,d,e])
            pos[(nx,ny)] = pos.get((nx,ny),0) + 1

    new_atoms = []
    for x,y,d,e in moved:
        if pos[(x,y)] >= 2: ans += e   # 충돌 → 에너지 누적
        else: new_atoms.append([x,y,d,e])
    atoms = new_atoms
    if not atoms: break
```

## 📌 5653 줄기세포 배양

### 💡 핵심 아이디어
- 각 세포 상태: `cell[(x,y)] = [생명력, 태어난시간]`
- 활성화 조건: `생명력 + 태어난시간 + 1 == 현재시간` → 이때 4방향 번식
- 사망 조건: `생명력*2 + 태어난시간 < 현재시간`
- 번식 충돌 시 **생명력 높은 놈** 우선, `occupied` set으로 점유 관리

```python
occupied = set()
cell = {}  # (x,y) -> [생명력, 태어난시간]

for time in range(1, k+1):
    trans_cell = {}
    for (x,y),(life,born) in cell.items():
        if life*2 + born < time: continue       # 이미 죽은 놈
        if life + born + 1 == time:             # 활성화 직후 → 번식
            for dir in dirs:
                nx,ny = x+dir[0], y+dir[1]
                if (nx,ny) in occupied: continue
                if (nx,ny) not in trans_cell or trans_cell[(nx,ny)] < life:
                    trans_cell[(nx,ny)] = life

    for (nx,ny),life in trans_cell.items():
        cell[(nx,ny)] = [life, time]
        occupied.add((nx,ny))

ans = sum(1 for life,born in cell.values() if life*2+born > k)
```
